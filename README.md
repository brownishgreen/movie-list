# Movie List

![封面截圖](/movie-list.png)

運用 Node.js 建立本機伺服器，並透過 Express 與 Template Engine (Handlebars) 建立簡易的電影清單網頁。

## Features
-瀏覽電影清單
- 瀏覽電影細節
- 以關鍵字搜尋電影

## RTE
Node.js

## Tools

- 開發環境：Visual Studio Code
- 應用程式框架：Express v4.19.2
- 樣版引擎：Express-Handlebars v7.1.3
- 樣式框架：Bootstrap v5.3.3
